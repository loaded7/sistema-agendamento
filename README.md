# 📅 Sistema de Agendamentos

![Screenshot do projeto](link-para-uma-imagem-do-projeto-funcionando)

> Sistema completo de agendamentos com backend em Node.js e frontend em React. Permite criar, listar, editar e cancelar agendamentos de forma simples e intuitiva.

## 🚀 Tecnologias Utilizadas

- **Backend:** Node.js, Express, CORS, Dotenv
- **Frontend:** React, Vite, Axios
- **Estilização:** CSS Modules
- **Ferramentas:** Git, GitHub, Insomnia (testes de API)

## 📋 Funcionalidades

- ✅ Listar todos os agendamentos
- ✅ Criar um novo agendamento
- ✅ Editar um agendamento existente
- ✅ Excluir um agendamento (com confirmação)
- ✅ Interface responsiva e amigável
